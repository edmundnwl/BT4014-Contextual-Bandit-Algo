# Adaptive LLM Routing via Contextual Bandits: Balancing Quality and Cost

### Dataset: The original dataset used for this project can be found on huggingface: https://huggingface.co/datasets/lmsys/chatbot_arena_conversations

### How to run
The package requirements can be found from requirements.txt, the order of which to run the files used for the final project implementation is as follows:
1. DA.ipynb
2. dueling_bandit_training.ipynb
3. contextual_bandit_routing_20_arm.ipynb
